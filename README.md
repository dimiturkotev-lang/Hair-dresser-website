# Elegant Salon - Hairdresser Website

A professional and modern hairdresser website built with HTML and CSS, featuring a responsive design and elegant styling.

## Project Overview

This website showcases a professional hair salon with a clean, user-friendly interface. It includes information about services, a welcoming hero section, and contact details.

## Features

- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- **Navigation Menu** - Easy-to-use navigation bar for quick access to different sections
- **Hero Section** - Eye-catching welcome section with call-to-action button
- **Services Showcase** - Grid layout displaying key hair services
- **About Us** - Information about the salon
- **Contact Section** - Contact information for customers
- **Professional Styling** - Purple-themed color scheme with modern CSS animations

## File Structure

```
Hair-dresser website/
├── index.html          # Main HTML file with website structure
├── styles.css          # CSS styling and responsive design
└── README.md           # Project documentation (this file)
```

## Getting Started

1. **Open the website**
   - Simply open `index.html` in any modern web browser

2. **View the website**
   - Navigate through different sections using the menu
   - Click "Book an Appointment" button in the hero section
   - Scroll to view all sections (Home, Services, About Us, Contact)

## Technologies Used

- **HTML5** - Semantic markup for proper structure
- **CSS3** - Modern styling with flexbox and grid layouts
- **Responsive Design** - Mobile-first approach with media queries
